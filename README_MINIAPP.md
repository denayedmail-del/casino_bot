# Casino Bot Mini-App

Цей додаток повністю функціональний Mini-App з гарним інтерфейсом та інтеграцією з БД.

## Налаштування

1. **Локальне тестування:**
   - Встановіть Python HTTP сервер: `python -m http.server 8000` в папці `web_app`.
   - Запустіть API: `uvicorn main:app --host 0.0.0.0 --port 8001` (в основній папці проекту).
   - Використовуйте ngrok для веб-додатку: `ngrok http 8000`.

2. **Хостинг веб-додатку:**
   - Завантажте папку `web_app` на веб-сервер з HTTPS (наприклад, Vercel, Netlify).
   - Отримайте URL, наприклад, `https://your-domain.com/web_app/index.html`.

3. **Запуск API на сервері:**
   - На GCP або іншому сервері запустіть бота та API: `uvicorn main:app --host 0.0.0.0 --port 8001`.
   - Змініть API_BASE в script.js на ваш сервер URL.

4. **Налаштування бота:**
   - У BotFather встановіть Web App URL: `/setmenubutton` -> URL веб-додатку.
   - Або додайте кнопку в повідомленнях.

5. **Запуск бота:**
   - Запустіть `python main.py` як звичайно.

## Функціонал

- **Гарний інтерфейс:** Кнопки для всіх дій, форми для введення.
- **Інтеграція з БД:** Баланс, токени оновлюються динамічно.
- **Повністю функціональні дії:**
  - Створення токенів (з витратою коштів).
  - Купівля/продаж токенів (з перевіркою балансу та оновленням постачання).
  - Ігри: дуель з користувачем, гра проти бота, пограбування.
  - Магазин: купівля товарів (VIP, бустер, захист).
  - Give гроші (тільки для адміна).
  - Топ гравців, допомога.

## Тестування

- Відкрийте бота в Telegram.
- Натисніть кнопку "Відкрити Mini-App".
- Взаємодійте з інтерфейсом – всі дії виконуються через API та оновлюють БД.
