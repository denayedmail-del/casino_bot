# Casino Bot Mini-App

Цей додаток перетворено в Telegram Mini-App.

## Налаштування

1. **Локальне тестування:**
   - Встановіть Python HTTP сервер: `python -m http.server 8000` в папці `web_app`.
   - Використовуйте ngrok: `ngrok http 8000` для HTTPS URL.

2. **Хостинг веб-додатку:**
   - Завантажте папку `web_app` на веб-сервер з HTTPS (наприклад, Vercel, Netlify, або власний сервер).
   - Отримайте URL, наприклад, `https://your-domain.com/web_app/index.html`.

3. **Налаштування бота:**
   - У BotFather встановіть Web App URL: `/setmenubutton` -> URL вашого веб-додатку.
   - Або додайте кнопку в повідомленнях.

4. **Запуск бота:**
   - Запустіть `python main.py` як звичайно.

## Функціонал

- Кнопки для основних команд.
- Поле для введення власних команд.
- Дані надсилаються боту через Web App API.

## Тестування

- Відкрийте бота в Telegram.
- Натисніть кнопку "Відкрити Mini-App".
- Взаємодійте з інтерфейсом.
